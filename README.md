# 💰 Personal Expense Tracker

A beautiful and responsive Personal Expense Tracker built using HTML, CSS, and JavaScript.

## ✨ Features

- Add Income
- Add Expenses
- Delete Transactions
- Live Balance Calculation
- Modern Dashboard
- Responsive Design
- Dark Theme UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
